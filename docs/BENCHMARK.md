# 🧪 Benchmark Module

## Visão Geral

O Benchmark Module mede a qualidade do RAG e do roteamento de especialistas.

## Campos do Caso de Teste

- Pergunta
- Especialista esperado
- Documento esperado
- Keywords esperadas
- Confiança esperada

## Critérios

- Especialista correto
- Documento correto
- Chunks relevantes
- Citação presente
- Confiança adequada
- Baixo risco de alucinação

## Resultado

- ✅ PASS
- ⚠️ WARNING
- ❌ FAIL

## Objetivo

Validar se o Guardião IA recupera o documento certo, escolhe o especialista correto e gera resposta baseada em evidências.
